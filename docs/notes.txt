Masking and Cropping

crop_rect(img,loc)

Returns a version of image img cropped to a square around the specified bundle location. loc is tuple of (centreX, centreY, radius).


         
get_mask(img, loc)
Returns a mask image with value of 1 inside bundle and 0 outside bundle based on specified bundle location. Returned image is same size as input image img. loc is tuple of (centreX, centreY, radius).


          
apply_mask(img, mask)
Applies a mask generated by get_mask to an image, returning the masked image.


        
auto_mask(img, [filterSize])     
Locates the bundle in an image and applies a mask to set pixel values outside of bundle to 0.
For optional parameter filterSize see find_bundle.

   
     
auto_mask_crop(img, [filterSize])
Locates the bundle in an image, applies a mask to set pixel values outside of bundle to 0, and then returns an image cropped to a square around the bundle.
For optional parameter filterSize see find_bundle.
